# Contributing to DaoSearch

Hey, thanks for wanting to help out. DaoSearch is a side project and we keep things pretty informal — no rigid process, no lengthy review cycles. If you want to build something, just go for it.

## Getting started

1. Fork the repo and clone it
2. Follow the setup in [README.md](README.md)
3. Pick something to work on — check [Issues](https://github.com/daosearchorg/daosearch/issues) or scratch your own itch

## Making changes

- Create a branch off `main`
- Keep commits focused — one thing per PR is easier to review
- Test your changes locally before opening a PR
- Write a short description of what you changed and why

## What we're looking for

- Bug fixes (always welcome)
- New features that make novel discovery better
- Performance improvements
- Better mobile experience
- Discord bot commands
- API integrations and tooling
- Translation improvements
- UI/UX tweaks

## What we're not looking for

- Major architectural rewrites without discussion first
- Changes that break existing functionality
- Features that turn this into a reading site (we're discovery only)

## Questions?

Open an issue or hop into the [Discord](https://discord.gg/Gmd3JXDuEU). We're pretty responsive there.
